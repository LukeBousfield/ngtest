Tag Syntax

Tags, as we talked about earlier, are marked with a less than angled bracket(<) and a greated than angled bracket(>)

Inside of the < key and the > key, goes the tag name. For example, the <html> tag tells the computer that the code is HTML. You end tags with </html>

Attribute syntax

Attributes, as we talked about earlier, are added to tags like this: <html lang="en">

Attributes go between the <html and the > with a space between <html and the attribute. After the space, specify an attribute, such as lang, then put an equals sign, then put what you want to set the attribute to, for example, en, in quotes, e.g. "en".
